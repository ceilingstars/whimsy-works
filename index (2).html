📱 Install & Sync Canopy

🤖 Android — Install as App

1. Open this file in Chrome
2. Tap ⋮ menu → Add to Home screen
3. Or go to pwabuilder.com, upload the canopy folder → download a real APK

🍎 iPhone — Install as App

Open in Safari → Share → Add to Home Screen

🖥️ Windows — Install as App

Open in Edge or Chrome → click ⊕ in address bar → Install Canopy

☁️ Sync across devices

Host the canopy folder on GitHub Pages (free, 5 min setup):
1. Create a free GitHub account
2. Upload the canopy folder as a new repo
3. Enable Pages in repo settings
4. Open that URL on any device — all your data syncs via the same browser!
